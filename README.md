Учебный проект по курсу «Прототипирование и вёрстка».  
Выполнены Задание 1 (прототип) и Задание 2 (вёрстка каталога).

---

libraria/

## Задание 1 — Прототип

**Тематика:** книжный интернет-магазин.

### Фильтры

| Фильтр | Тип элемента | Логика |
|--------|-------------|--------|
| Жанр | Чекбоксы | Множественный выбор |
| Цена (₽) | Двойной ползунок + числовые поля | Диапазон от–до |
| Рейтинг | Радиокнопки | Один вариант (минимальная оценка) |
| Язык издания | Чекбоксы | Множественный выбор |
| Год издания | Чекбоксы | Множественный выбор |

### Расположение элементов
- **Header** — sticky, всегда доступен для поиска.
- **Hero-баннер** — заголовок каталога, создаёт контекст.
- **Sidebar слева** — стандарт e-commerce; не перекрывает сетку товаров.
- **Активные теги фильтров** — над сеткой, явная обратная связь о применённых фильтрах.
- **Карточки** — `auto-fill` grid, адаптируются под ширину экрана.
- **Пагинация** — кнопка «Показать ещё» + номера страниц.

### Тип фильтрации
Клиентская фильтрация с `debounce` для мгновенного отклика при изменении чекбоксов и ползунка. При первой загрузке параметры передаются через URL (`?genre=classics&price_max=2000`), что обеспечивает SEO и возможность поделиться ссылкой.

### Адаптив
| Ширина | Поведение |
|--------|-----------|
| > 900 px | Двухколоночный layout: sidebar + сетка карточек |
| ≤ 900 px | Sidebar скрыт; появляется кнопка «Фильтры» — тап открывает его |
| ≤ 600 px | Поиск на всю ширину, кнопки корзины/входа скрыты, карточки от 160 px |

## Задание 2 — Вёрстка

### Технологии
- Чистый **HTML5 / CSS3 / Vanilla JS** (без фреймворков).
- Шрифты: **Playfair Display** (заголовки) + **DM Sans** (текст) — Google Fonts.

### Что реализовано

#### Семантическая вёрстка
`<header>`, `<aside>`, `<main>`, `<article>` для каждой карточки, `<footer>`.

#### Карточки товара (8 штук)
Каждая карточка содержит:
- Обложку-плейсхолдер с уникальным цветовым градиентом
- Жанровую метку, название, автора
- Звёздный рейтинг с количеством отзывов
- Текущую цену и (при скидке) зачёркнутую старую
- **Кнопку лайка** ♡ / ♥ — переключается кликом
- **Поле комментария** — textarea с placeholder
- Кнопку «В корзину»

#### Фильтры (5 штук)
Все интерактивны: чекбоксы отмечаются, радиокнопки переключаются, ползунок двигается, числовые поля синхронизированы.

#### Визуальные детали
- CSS-переменные для всей цветовой схемы
- Тени, скругления, hover-эффекты на всех элементах
- Появление карточек с анимацией `fadeUp` + staggered delay
- Активные теги фильтров с анимированным удалением
- Sticky header с backdrop-blur у кнопки лайка

## Как запустить

Просто откройте `index.html` в браузере — никакого сервера не требуется.

```bash
# Или через простой HTTP-сервер:
python3 -m http.server 8080
# Откройте http://localhost:8080
```
